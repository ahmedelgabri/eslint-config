# `@ahmedelgabri/eslint-config`

My personal set of eslint rules

## Install

```sh
$ npm i -D @ahmedelgabri/eslint-config 
```

```json
{
  "extends": ["@ahmedelgabri"]
}
```

You might need to install some peerDependencies

```sh
$ npm i -D eslint@3.19.0 eslint-config-prettier@2.1.1 eslint-import-resolver-webpack@0.8.1 eslint-plugin-import@2.3.0 eslint-plugin-jest@20.0.3 eslint-plugin-jsx-a11y@5.0.3 eslint-plugin-prettier@2.1.1 eslint-plugin-react@7.0.1 prettier@1.4.4 webpack@2.6.1
```
