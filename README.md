# `@ahmedelgabri/eslint-config`

My personal set of eslint rules

## Install

```sh
npm i -D eslint @ahmedelgabri/eslint-config
```

```json
{
  "extends": ["@ahmedelgabri"]
}
```

## Extra

```
{
  "extends": ["@ahmedelgabri", "@ahmedelgabri/eslint-config/<config name>"]
}
```
